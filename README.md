# caRbs

<!-- badges: start -->

<!-- badges: end -->

The caRbs package is designed to compute carbonate budgets for coral
reef locations across the Great Barrier Reef (GBR).

## Installation

You can install the development version of caRbs like so:

``` r
devtools::install_github("aadesbiens/caRbs")
```
